## Keybindings

Only some of the useful ones.

|Keybinding|Action|
|---|---|
|`?`|help menu|
|`shift+d`|open reset options menu|
|`/`|search in current pane|
|`1` to `5`|cycle panes|
|`+` and `_`|cycle window sizes (change splits)|
