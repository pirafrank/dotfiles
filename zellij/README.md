## Setup

Symlink config

```sh
ln -s ~/dotfiles/zellij/config.kdl ~/.config/zellij/config.kdl
```

or set env var `ZELLIJ_CONFIG_DIR` to a dir with structure to replace `~/.config/zellij`.
